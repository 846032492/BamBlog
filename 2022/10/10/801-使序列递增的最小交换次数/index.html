801. 使序列递增的最小交换次数

我们有两个长度相等且不为空的整型数组 nums1 和 nums2 。在一次操作中，我们可以交换 nums1[i] 和 nums2[i]的元素。

  • 例如，如果 nums1 = [1,2,3,8] ， nums2 =[5,6,7,4] ，你可以交换 i = 3 处的元素，得到 nums1 =[1,2,3,4] 和 nums2 =[5,6,7,8] 。

返回 使 nums1 和 nums2 严格递增 所需操作的最小次数 。

数组 arr 严格递增 且 arr[0] < arr[1] < arr[2] < … < arr[arr.length - 1] 。

注意：

  • 用例保证可以实现操作。

示例 1:

输入: nums1 = [1,3,5,4], nums2 = [1,2,3,7]
输出: 1
解释:
交换 A[3] 和 B[3] 后，两个数组如下:
A = [1, 3, 5, 7] ， B = [1, 2, 3, 4]
两个数组均为严格递增的。

示例 2:

输入: nums1 = [0,3,5,8,9], nums2 = [2,1,4,6,9]
输出: 1

提示:

  • 2 <= nums1.length <= 10^5
  • nums2.length == nums1.length
  • 0 <= nums1[i], nums2[i] <= 2 * 10^5

思路

因为题目说明了用例保证可以实现操作,且只能相同位置的交换

分析情况:

  • 情况1、nums1[i] > nums1[i-1] && nums2[i] > nums2[i-1]
  • 情况2、nums1[i] > nums2[i-1] && nums2[i] > nums1[i-1]

因为每次当前的交换次数都和上一次的相关

我们可以用二维数组 dp[i] [2]，二维数组的存放0,1；分别代表不交换和交换

dp[i] [0]的值代表当前位置不交换，且总共的操作次数，

dp[i] [1]的值代表当前位置交换，且总共的操作次数

会出现三种情况:

  1. 上述情况1：当前不交换、上一个位置也不交换；当前交换、上一个也交换
  2. 上述情况2：当前不交换、上一个位置交换；当前交换、上一个不交换
  3. 情况1 和 情况2 并存：都可以交换，拿到最小值

初始状态：dp[0] [0] = 0 , do[0] [1] ; 数组首位要么交换，要么不交换

转移方程：

  • 情况1：dp[i] [0] = dp[i-1] [0]，dp[i] [1] = dp[i-1] [1] + 1
  • 情况2：dp[i] [0] = dp[i-1] [1]，dp[i] [1] = dp[i-1] [0] + 1
  • 情况3：dp[i] [0] = Math.min( dp[i-1] [0] , dp[i-1] [1]) ，dp[i] [1] = dp[i] [0] + 1

优化：

因为都只需要上一次状态的值，所以不需要全部记录，可以使用滚动数组
