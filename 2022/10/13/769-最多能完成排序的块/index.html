769. 最多能完成排序的块

给定一个长度为 n 的整数数组 arr ，它表示在 [0, n - 1] 范围内的整数的排列。

我们将 arr 分割成若干 块 (即分区)，并对每个块单独排序。将它们连接起来后，使得连接的结果和按升序排序后的原数组相同。

返回数组能分成的最多块数量。

提示:

  • n == arr.length
  • 1 <= n <= 10
  • 0 <= arr[i] < n
  • arr 中每个元素都 不同

思路

方法一：模拟

因为题目给了n数组中 0 <= arr[i] < n，我们只需要找出每个最大值所在的索引位置,并计数统计

方法二：单调栈

找出左边界和右边界，每次和栈顶元素作比较

  • 栈顶元素 < 未入栈的数组值，则可以直接入栈

  • 栈顶元素 > 未入栈的数组值

  1. 弹出栈顶元素并记录、并且再次和栈顶元素比较，多次循环
  2. ……..
  3. 直到栈顶元素 < 未入栈的值，最后需要把一开始记录的栈顶最大元素添加回栈

返回栈顶元素个数
