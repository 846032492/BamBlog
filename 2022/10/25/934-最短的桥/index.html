934. 最短的桥

给你一个大小为 n x n 的二元矩阵 grid ，其中 1 表示陆地，0 表示水域。

岛 是由四面相连的 1 形成的一个最大组，即不会与非组内的任何其他 1 相连。grid 中 恰好存在两座岛 。

你可以将任意数量的 0 变为 1 ，以使两座岛连接起来，变成 一座岛 。

返回必须翻转的 0 的最小数目。

提示：

  • n == grid.length == grid[i].length
  • 2 <= n <= 100
  • grid[i][j] 为 0 或 1
  • grid 中恰有两个岛

思路

先找到第一个岛屿，并且使用深度优先搜索把整个岛屿搜索出来并设置为2代表搜索过，在搜索过程中若遇到周边海域0的情况下则放入一个队列中进行保存；

遍历队列中所存储的海域，进行填海造陆，依次向外层扩展，直到遇到下一个海岛的情况
