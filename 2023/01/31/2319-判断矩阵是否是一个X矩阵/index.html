2319. 判断矩阵是否是一个 X 矩阵

如果一个正方形矩阵满足下述 全部 条件，则称之为一个 X 矩阵 ：

  1. 矩阵对角线上的所有元素都 不是 0
  2. 矩阵中所有其他元素都是 0

给你一个大小为 n x n 的二维整数数组 grid ，表示一个正方形矩阵。如果 grid 是一个 X 矩阵 ，返回 true ；否则，返回 false 。

示例 2：

提示：

  • n == grid.length == grid[i].length
  • 3 <= n <= 100
  • 0 <= grid[i][j] <= 105

思路

根据X矩阵特性可以发现矩阵的对角线满足

  • i == j || i + j == n - 1

n表示矩阵长度、i 表示行下标索引、j表示列下标索引
